# 📊 Telco Customer Churn Analytics

An end-to-end Data Analytics and Machine Learning project that analyzes customer churn in the telecommunications industry. The project includes data preprocessing, exploratory data analysis (EDA), predictive modeling, and an interactive Power BI dashboard to uncover key business insights and support data-driven decision-making.

---

## 📌 Overview

Customer churn is one of the most critical challenges faced by telecom companies. Understanding why customers leave enables businesses to improve customer satisfaction and implement effective retention strategies.

This project focuses on:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Customer churn prediction using Machine Learning
- Interactive Power BI dashboard
- Business insights and recommendations

## 📊 Exploratory Data Analysis

Key findings from the analysis include:

- Customers with **Month-to-Month** contracts are significantly more likely to churn.
- **Fiber Optic** users exhibit higher churn rates compared to DSL users.
- Customers using **Electronic Check** as their payment method have the highest churn probability.
- Customers with **One-Year** and **Two-Year** contracts demonstrate better retention.
- Higher monthly charges are associated with increased churn.

## 💡 Business Recommendations

- Encourage customers to switch from month-to-month contracts to long-term plans.
- Improve customer experience for Fiber Optic subscribers.
- Design targeted retention campaigns for customers identified as high-risk.
- Offer incentives for customers using electronic check payment methods.
- Leverage predictive analytics for proactive customer retention.
